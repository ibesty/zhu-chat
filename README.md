# zhu-chat

模仿微信网页版写的聊天应用，基于Vue和socket.io
> A chat program for zhuzhu

[screenshot](https://github.com/ibesty/zhu-chat/raw/master/screenshot/screenshot.png)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

#run server
cd server
npm start
```
